# @lemon/image-manager

Obsidian vault의 이미지를 자동으로 WebP 포맷으로 변환하여 저장 용량을 줄여주는 플러그인입니다.

## 기능

- **자동 변환**: 이미지를 vault에 추가하거나 클립보드에서 붙여넣기 할 때 자동으로 WebP로 변환
- **링크 자동 업데이트**: 변환 후 마크다운 문서의 이미지 링크(`![[...]]`, `![](...)`)를 자동으로 갱신
- **원본 삭제**: 변환 완료 후 원본 파일 자동 삭제
- **품질 설정**: WebP 변환 품질을 1~100 사이에서 조절 가능 (기본값: 80)

### 지원 포맷

| 입력 | 출력 |
|---|---|
| JPG / JPEG | WebP |
| PNG | WebP |
| BMP | WebP |
| TIFF / TIF | WebP |

> GIF(애니메이션), SVG(벡터)는 변환 대상에서 제외됩니다.

## 설치

1. Obsidian 설정 → 커뮤니티 플러그인 → 직접 설치
2. `manifest.json`, `main.js`, `styles.css`를 vault의 `.obsidian/plugins/lemon-image-manager/` 폴더에 복사
3. 플러그인 활성화

## 설정

| 항목 | 설명 | 기본값 |
|---|---|---|
| 자동 변환 | 이미지 추가 시 자동 변환 여부 | ON |
| WebP 품질 | 변환 품질 (높을수록 고화질, 파일 크기 증가) | 80 |

## 요구사항

- Obsidian 1.0.0 이상
- 데스크탑 전용 (Windows / macOS / Linux)
