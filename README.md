# @lemon/image-manager

Obsidian vault의 이미지를 자동으로 WebP 포맷으로 변환하여 저장 용량을 줄여주는 플러그인입니다.

## 기능

- **자동 변환**: 이미지를 vault에 추가하거나 클립보드에서 붙여넣기 할 때 자동으로 WebP로 변환
- **일괄 변환**: `Ctrl+P` → "볼트 내 이미지 일괄 변환" 명령으로 기존 이미지 전체 변환
- **용량 비교 알림**: 변환 전후 용량 및 감소율 표시 (예: `1.2MB → 340KB (-72%)`)
- **링크 자동 업데이트**: 변환 후 마크다운 문서의 이미지 링크(`![[...]]`, `![](...)`)를 자동으로 갱신
- **원본 삭제**: 변환 완료 후 원본 파일 자동 삭제
- **품질 설정**: WebP 변환 품질을 1~100 사이에서 조절 가능 (기본값: 80)
- **제외 폴더 설정**: 특정 폴더는 변환 대상에서 제외 가능

### 지원 포맷

| 입력 | 출력 |
|---|---|
| JPG / JPEG | WebP 또는 AVIF (설정에서 선택) |
| PNG | WebP 또는 AVIF |
| BMP | WebP 또는 AVIF |
| TIFF / TIF | WebP 또는 AVIF |
| HEIC / HEIF | WebP 또는 AVIF |

> GIF(애니메이션), SVG(벡터)는 변환 대상에서 제외됩니다.

### 출력 포맷 비교

| | WebP | AVIF |
|---|---|---|
| 압축률 | 좋음 | 더 좋음 |
| 변환 속도 | 빠름 (Canvas API) | 느림 (WASM, 첫 실행 시 초기화) |
| 호환성 | 넓음 | Obsidian 1.0+ |

## 설치

1. Obsidian 설정 → 커뮤니티 플러그인 → 직접 설치
2. `manifest.json`, `main.js`, `styles.css`를 vault의 `.obsidian/plugins/lemon-image-manager/` 폴더에 복사
3. 플러그인 활성화

## 설정

| 항목 | 설명 | 기본값 |
|---|---|---|
| 자동 변환 | 이미지 추가 시 자동 변환 여부 | ON |
| 출력 포맷 | WebP 또는 AVIF | WebP |
| 변환 품질 | 변환 품질 (높을수록 고화질, 파일 크기 증가) | 80 |
| 제외 폴더 | 변환하지 않을 폴더 경로 (한 줄에 하나씩) | 없음 |

## 요구사항

- Obsidian 1.0.0 이상
- 데스크탑 전용 (Windows / macOS / Linux)
